# 作品說明

2022 The F2E week1 - The F2E 活動網站設計

# 系統說明

首先先

```
git clone https://github.com/Beginneraboutlife116/the-f2e-2022-week1.git
```

```
cd the-f2e-2022-week1
```

```
npm install
```

```
npm start
```
